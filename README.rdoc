== Ext.ux.touch.GridSummary

A plugin to add a summary dock or row to the Ext.ux.TouchGridPanel

Possible options:<br>
-type<br>
--dock (default)<br>
--row<br>
-summaryType<br>
--total (default)<br>
--average<br>
-decimalPosition<br>
--Any number, default is 2

= Usage

   {
       xtype: 'touchgridpanel',
       ...,
       plugins: [new Ext.ux.touch.GridSummary()],
       ...
   }
